) const

Test si un rayon traverse la boite.
Si c'est le cas, la methode retournera les distance d'intersection sur la boite. (cf. Rayon)
Utilise la méthode intersectIn().

Paramètres
[in]rUn rayon quelconque. (cf. Rayon)
[out]distanceMincoefficient de distance du premier impacte sur la boite par rapport à l'origine du rayon.
Le coefficient est dependant de la norme du vecteur direction du rayon.
[out]distanceMaxcoefficient de distance du second impacte sur la boite par rapport à l'origine du rayon.
Le coefficient est dependant de la norme du vecteur direction du rayon.
Renvoie
Le resultat du test. True = intersection, false = pas d'intersection.
