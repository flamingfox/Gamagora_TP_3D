bool Terrain::intersect ( const Rayon &  rayon,
float &  coeffDistance,
int &  i 
) const

Test si un rayon traverse le terrain.
Si c'est le cas, la methode retournera la distance d'intersection la plus proche sur le terrain. (cf. Rayon)
Utilise la méthode intersect() de Box.

Paramètres
[in]rayonUn rayon quelconque. (cf. Rayon)
[out]coeffDistancecoefficient de distance de l'impacte sur le terrain par rapport à l'origine du rayon.
Le coefficient est dependant de la norme du vecteur direction du rayon.
[out]Nombred'itération durant le ray marching.
Renvoie
Le resultat du test. True = intersection, false = pas d'intersection.
